Detailed Description

The base economic strategy for turkey domestication.

This class calculates the number of turkey needed by a household, and forces the household to track the number of turkeys they keep.

The execute method controls these actions.

Domestication itself is toggled in the Village class.

int com.mesaverde.domestication.StrictEconomicModel.execute ( long  hh_protein_need)

A method for controlling turkey domestication. This methos is fed

hh_protein_need

as a parameter, calculates the number of turkeys needed to fulfull that need, and compares that with the maximum number of turkeys allows so as not to deplete maize stores beyond a critical threshold (the amount of maize utilized in the past year). The agent will choose to domesticate the lesser of these two turkey amounts.
